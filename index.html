Visitor Lookup

This app consolidates and presents site visitor information from multiple closed systems, making it easily accessible to all site staff. It serves as a launch page for other internal applications accessing visitor information.

  • Used By :All staff
  • Technology :Dotnet 8 / Blazor
  • Systems :INX InFlight, Gallagher, WorkDay
  • Access :Internal - Restricted

Features

  • Keyword lookup by name, company, department, or job title
  • Date range search to view all visitors within a specific period
  • Contractor visit plan to show expected contractor numbers for a period

Key Benefits

Integrates visitor data from INX InFlight, INX Intuition, Gallagher, and Workday. Providing information on:

  • Onsite times
  • Flight and accommodation bookings
  • Work group and site contact
  • Induction and compliance status
  • Emergency contacts
  • Past and future roster plan

"Before we had the Visitor Lookup app we had to constantly send emails to the Travel team to confirm our contractor bookings. This app was a huge producitivty boost for our supervisors."

Services, Superintendent

QR Compliance Check

This system introduces QR codes to all site access cards, linking to each individual's compliance status. It enables live, in-field spot checks of a person's compliance at any time. Accessible directly over the internet, the app can also be used by external contractors.

  • Used By :All Site Workers
  • Technology :Dotnet 8
  • Systems :INX InTuition / Gallagher
  • Access :Internal - authenticated users

Overview

The app categorises compliance into three sections:

  • Inductions: General site access and area inductions
  • Role: Compliances required for the specified role
  • Individual: Additional competencies obtained by the individual

INX InTuition

The QR Compliance Check app displays information set up in INX InTuition. Significant support was provided to help the business correctly structure site compliance policies in INX.

"In November 2023 we had a large shutdown with over 700 contractors. The QR compliance checker allowed us to capture and verify that work was occurring in a compliant manner. The contractor supervisors able to do the checks themselves was a huge help!"

Utilities, Superintendent

Evacuation Mustering

Integrates with the Gallagher API to provide a live evacuation and mustering interface.

Allows members of the Incident Management Team the ability to closely monitor the progress of an evacuation and also to manually update personnel locations.

This is displayed on a large TV in the Incident Management control room.

  • Used By :IMT / Security
  • Technology :Dotnet 8
  • Systems :Gallagher
  • Access :Internal - Restricted

Features

  • Integrates directly with the Gallagher API
  • Live status of which zones all personnel are in
  • Simple interface to filter by zones to identify priority personnel to evacuate
  • Live feed of personnel zone changes
  • Bulk changing of perssonnel zones

Employee Attendance

Integrates employee roster plan from INX InFlight with card swipe events from Gallagher to generate a planned timesheet vs actual timesheet for all employees.

Includes an automatic notification system for unexpectred absence alerts.

As well as a leadership reporting process for managing ongoing 'Missing Days' where an absence is not reconciled with an approved leave day from Work Day.

  • Used By :Leaders / Payroll / HR
  • Technology :.NET Framework
  • Systems :INX InFlight / Gallagher / WorkDay
  • Access :Internal - Restricted

Features

  • Pulls roster plan for INX InFlight and site records from Gallagher
  • Handles logic for night shift, flight days, unplanned roster changes, toil days
  • Comprehensive system for reporting on missing days
  • Integrates approved leave from Work Day

ICT COST Control

This proof-of-concept system imports ICT transactions from Pronto, allowing direct linkage to budget and forecast items. It enabled unparalleled cost accountability by reporting expenses per budget item instead of GL codes and has been successfully used for three consecutive years.

  • Used By :ICT Site Lead
  • Technology :Dotnet 8
  • Systems :Pronto
  • Access :Internal - Restricted

Features

  • Simple interface to link each ICT transaction to a budget item
  • Introduced the metric of 'Planned vs Unplanned' expenditure
  • Serves as the data source for comprehensive Power BI reports

Radio Management

At Ravensthorpe, all radio devices were leased, making it essential for ICT to account for radios allocated in the field. The radio management app has been invaluable in ensuring these assets are properly tracked.

  • Used By :All Employees
  • Technology :.NET Framework
  • Systems :Motorola TRBONet / WorkDay
  • Access :Internal - Restricted

Features

  • Technology :Power Bi
  • Systems :Pronto / INX
  • Access :Power Bi Cloud

Features

INX to Gallagher Sync

This important integration synchronises data from INX into Gallagher by leveraging the Gallagher HTTP API.

The integration allows for the automatic creation of site access cards based on INX InFlight profile and roster data, ensuring that all visitor information is managed in a single location (INX InFlight) and is consistent across both systems without manual data management.

In addition to cardholder information, site compliance details are also synchronised from INX InTuition, ensuring that only compliant personnel are granted site access.This integration has been a key part of the site's security and compliance strategy.

  • Used By :System Integration
  • Technology :Power Shell
  • Systems :INX / Gallagher

All personal data fields populated from INX InFlight

Cardholder competencies populated from INX InTuition - Site Induction
